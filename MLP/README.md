# Multi Layer Perceptron

This experiment implements a **Neural Network** using the Fashion MNIST dataset. Learn image preprocessing, flattening, model construction, training, evaluation, and automated hyperparameter optimization.


## Dataset

The dataset Fashion MNIST contains **60000 training samples** and **10000 testing samples**, each sample is a 28 x 28 image with 10 different classes/ variety of clothes.


---

## Requirements

Install the required Python packages using:

```bash
pip install -r requirements.txt
```

- pandas
- numpy
- matplotlib
- seaborn
- scikit-learn
- tensorflow
